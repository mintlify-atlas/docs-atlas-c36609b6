<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your portfolio up and running in minutes
  </p>

  <Steps>
    <Step title="Clone the repository">
      Start by cloning the Astro Portfolio template to your local machine:
      
      ```bash
      git clone https://github.com/Hazielgmz/astro-Portfolio.git
      cd astro-Portfolio
      ```
    </Step>

    <Step title="Install dependencies">
      Install the required dependencies using pnpm (or npm/yarn):
      
      <CodeGroup>
      ```bash pnpm
      pnpm install
      ```
      
      ```bash npm
      npm install
      ```
      
      ```bash yarn
      yarn install
      ```
      </CodeGroup>
    </Step>

    <Step title="Configure Supabase">
      Create a `.env` file and add your Supabase credentials:
      
      ```bash
      SUPABASE_URL=your_supabase_url
      SUPABASE_KEY=your_supabase_key
      ```
      
      <Note>
        You'll need to set up your Supabase database with the required tables. See the [Supabase Setup](/configuration/supabase) guide for details.
      </Note>
    </Step>

    <Step title="Start the development server">
      Run the development server and view your portfolio:
      
      ```bash
      pnpm dev
      ```
      
      Open [http://localhost:4321](http://localhost:4321) in your browser to see your portfolio.
    </Step>
  </Steps>
</div>
