## Welcome to Astro Portfolio

Astro Portfolio is a modern, server-side rendered portfolio template designed for developers, designers, and creative professionals who want to showcase their work with a dynamic, database-driven approach.

## What is Astro Portfolio?

This template combines the power of **Astro's server-side rendering** with **Supabase's real-time database** to create a portfolio that's both performant and easy to manage. Unlike static portfolios that require code changes to update content, Astro Portfolio lets you manage your projects, career timeline, certifications, and tools directly from a Supabase database.

## Key Benefits

### Dynamic Content Management

Update your portfolio content without touching code. All your projects, career history, tools, and certifications are stored in Supabase and can be updated through a database interface.

### Server-Side Rendering

Astro's SSR ensures your portfolio loads fast and is SEO-friendly. Content is rendered on the server, providing optimal performance and search engine visibility.

### Modern Tech Stack

Built with cutting-edge technologies that developers love:

- **Astro** - Fast, modern static site generation with SSR
- **Tailwind CSS** - Utility-first CSS framework for rapid UI development
- **Supabase** - PostgreSQL database with real-time capabilities
- **Alpine.js** - Lightweight JavaScript for interactive components
- **TypeScript** - Type-safe development experience

### Beautiful, Responsive Design

The template features:

- Gradient backgrounds with blur effects
- Animated navbar that appears on scroll
- Smooth scroll animations
- Dark mode support
- Fully responsive layouts
- Interactive modals for detailed content

## What You'll Build

With this template, you get a complete portfolio website featuring:

- **Hero Section** - Dynamic about me section with profile image and bio
- **Career Timeline** - Professional experience displayed chronologically
- **Project Showcase** - Projects with images, descriptions, tech stacks, and links
- **Tools & Technologies** - Interactive display of your technical skills
- **Certifications** - Professional certificates linked to relevant technologies
- **Responsive Navigation** - Fixed header with scroll-based animations

<Note>
  All content is managed through Supabase, making it easy to update your portfolio without deploying code changes.
</Note>
